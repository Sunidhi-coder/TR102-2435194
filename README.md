Topics Covered :
<br>
-<u>Data:</u> Data is a collection of raw facts, figures, or observations that has not yet been processed.
<br>
-<u>Information:</u> Information is processed and organized data that is meaningful and useful for making decisions.
Eg: If we want to take all the datails of accidents on a particular Highway then our data is newspapers.
<br>
-<u>Data Sources:</u> Data sources are places from which data is collected.
    Common data sources:Databases,CSV/Excel files,Websites,Sensors and IoT devices,Social media,Surveys,APIs etc.
<br>
-<u>Types of data:</u> 
    Structured Data: Organized in rows and columns (e.g., SQL database).
    Unstructured Data: No predefined format (e.g., images, videos, emails).
<br>
<u>Data Exploration:</u> Data exploration is the process of examining data to understand its structure, identify patterns, detect missing values, and discover relationships before analysis or model building.
<br>
-<u>Data Modality:</u> Data modality refers to the type or format of data used in AI and machine learning.
     data representations: Text,Images,Audio,Video,Tabular data.
<br>
-<u>Machine Learning (ML) Model:</u> A Machine Learning model is an algorithm that learns patterns from data to make predictions or decisions without being explicitly programmed.
<br>
-<u>Generative AI (Gen AI):</u> Generative AI is a type of artificial intelligence that creates new content such as text, images, music, videos, or code by learning patterns from existing data.  Eg:ChatGPT (text generation),GitHub Copilot.
